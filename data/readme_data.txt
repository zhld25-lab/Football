ScoutAI data notes
==================

sample_players.csv is a small demonstration dataset created for this university project. It contains fictional but realistic youth football player records so the full pipeline can run immediately with python main.py.

Full datasets are not included in this repository. Large public football datasets should be downloaded from the original providers when needed:

1. Transfermarkt datasets:
   https://github.com/dcaribou/transfermarkt-datasets

2. Football Manager 2023 dataset:
   https://www.kaggle.com/datasets/siddhrajthakor/football-manager-2023-dataset

The full datasets are excluded because of file size, licensing, and data usage restrictions. Users should always review the license and terms of use for each source before using the data in coursework, research, or club analysis.

Data should be used responsibly. U18 players are minors, so this project should only use legally accessible public data and should not collect private medical records. Injury-related information, if used in future work, should be treated only as public availability indicators such as appearances, minutes played, missed matches, or publicly reported absence information.
